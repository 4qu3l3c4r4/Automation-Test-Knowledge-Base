Introduction

We are http://automated-testing.info community collects different test automation examples to help software engineers bootstrap test automation, learn new approaches\technologies and improve their solution as we are doing the same tasks day by day, so let's share our experience to get more from collective mind

Goal

Collect as much as possible different examples on programming, design patterns, framework usage, tools usage in context of test automation. If you know something about test automation on java python C# ruby php javascript webdriver selenium jmeter soapui robot framework visual studio telerik qtp testcomplete appium robotium jenkins sikuli web test automation desktop test automation mobile test automation enterprise test automation test automation infrastructure performance test automation .etc, feel free to contribute. Let's create mega knowledge base together!

Contribution Guidelines

  1. Fork knowledge base
